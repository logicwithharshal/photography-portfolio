# 📸 Photographer Portfolio Website

A modern and responsive photographer portfolio website built using HTML, CSS, and JavaScript.

Designed to showcase photography work with an elegant glassmorphism UI, interactive animations, image gallery, and professional portfolio sections.

## ✨ Features

* Modern Glassmorphism Design
* Fully Responsive Layout
* Animated Sidebar Navigation
* Hero Section with Typing Effect
* Beautiful Masonry Gallery
* Loading Screen Animation
* Statistics Section
* About Section
* Contact Form
* Social Media Icons
* Smooth Scrolling Navigation
* Back-to-Top Button
* Hover Animations & Transitions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome Icons
* Google Fonts (Inter)
